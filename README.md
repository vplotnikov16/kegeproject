# НАЗВАНИЕ_ИС

Основной скрипт для запуска проекта - run.py. Для запуска без ошибок нужно убедиться, что установлены все зависимости.
```bash
python run.py
```

Для удобного запуска и автоматической настройки окружения есть скрипт `run_in_venv.py`. 
```bash
python run_in_venv.py
```
Он создаёт виртуальное окружение `.venv` в корне проекта (если его нет), ставит недостающие библиотеки из `requirements.txt` и запускает `run.py` в этом окружении.
